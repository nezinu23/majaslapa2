# ĶLUS (3 formas) — JSON filtri + pieslēgšanās (papildus)

## Pamata prasība (skolai)
3. formā tiek ielādēti dati no:
- `vielas.json`
- `inventars.json`

Un ar pogām tiek filtrēts:
- Vielas
- Inventārs
- Rādīt visu

- 2. forma ļauj pieslēgties, izmantojot `users.json` (client-side).
- Ja NAV pieslēdzies: 3. formā redzi tikai pirmos 3 ierakstus.
- Ja esi pieslēdzies: redzi visus ierakstus + vari pievienot/dzēst savas rindas (saglabājas localStorage).
